MUSCLE CARS WEBSITE PROJECT
===========================

Project Structure:
- index.html — Home page with hero section, features, portfolio, and services
- contact.html — Contact form for quote requests
- css/styles.css — External stylesheet with responsive design
- images/ — All images (hero.jpg, project1.jpg, logo.png)
- assets/ — Additional assets (resume.pdf, etc.)

Features:
✓ Responsive design (desktop, tablet, mobile)
✓ Semantic HTML with accessibility features
✓ External CSS with CSS variables
✓ Internal CSS (example in index.html)
✓ Inline CSS (example on contact.html button)
✓ Contact form with validation
✓ Navigation menu
✓ SEO meta tags

To Run Locally:
1. Open this folder in VS Code (File → Open Folder)
2. Install Live Server extension (if not installed)
3. Right-click index.html → Open with Live Server
4. Site will open at http://127.0.0.1:5500/

CSS Types Used:
- External: css/styles.css
- Internal: <style> block in index.html
- Inline: style attribute on contact.html button

Accessibility Features:
- HTML lang="en" for language
- Alt text on all images
- Semantic tags (<header>, <main>, <section>, <footer>, <nav>)
- Form labels with matching IDs
- Proper meta tags for SEO
